  <action>
    Add dispose() method to AudioEngine class after the toggleMute() method (after line 1267).

    The dispose() method must:
    1. Stop all buffer sources (noiseNodes) using stop() - requires calling stop(0) on each
    2. Disconnect all nodes from their connections:
       - Disconnect each gain node from masterGain
       - Disconnect each filter from its gain
       - Disconnect each noise source from its filter
       - Disconnect masterGain from destination
    3. Clear all node references (set to null)
    4. Set initialized = false
    5. If ctx exists and is running, close it to release resources

    ```javascript
    dispose() {
      if (!this.initialized) return;

      // Stop all buffer sources
      Object.values(this.noiseNodes).forEach(node => {
        try { node.stop(); } catch(e) {}
      });

      // Disconnect and clear noise nodes
      Object.values(this.noiseNodes).forEach(node => {
        try { node.disconnect(); } catch(e) {}
      });
      this.noiseNodes = {};

      // Disconnect and clear filters
      Object.values(this.filters).forEach(filter => {
        try { filter.disconnect(); } catch(e) {}
      });
      this.filters = {};

      // Disconnect and clear gains
      Object.values(this.gains).forEach(gain => {
        try { gain.disconnect(); } catch(e) {}
      });
      this.gains = {};

      // Disconnect master gain
      if (this.masterGain) {
        try { this.masterGain.disconnect(); } catch(e) {}
        this.masterGain = null;
      }

      // Close audio context
      if (this.ctx) {
        try { this.ctx.close(); } catch(e) {}
        this.ctx = null;
      }

      this.initialized = false;
    }
    ```
